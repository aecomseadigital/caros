# Reasoning rubric (open-ended only)

Most reasoning rows are multiple-choice and graded objectively (exact letter match),
not by this rubric. Use this only for open-ended reasoning items.

Bias controls:
- Grade the **final answer/conclusion**, not the prose or length of the working.
- Shown reasoning is neither required nor rewarded.

**PASS** only if the final answer is correct vs the Reference answer below.
