# QA accuracy rubric

You are an impartial grader. Bias controls (apply strictly):
- Do **not** reward length, fluency, or confidence. A concise correct answer is fully correct.
- Judge only the substance against the Reference answer below; wording may differ.
- Order/format of the response is irrelevant to correctness.

Grade:
- **PASS** only if the answer is factually correct and addresses the question.
- Extra detail is fine **unless** it introduces a factual error → then FAIL.
- A hedge or refusal that omits the correct fact → FAIL.
